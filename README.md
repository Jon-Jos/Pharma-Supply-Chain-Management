# Pharma-Supply-Chain-Management-App 

Blockchain (Hyperledger Fabric) Supply Chain Management Dapp with smart contract written in Javascript.

## Rerequisites 
    
    Go version 1.14.x
    Docker and Docker Compose
    Node.js runtime and npm : 12.22.12
